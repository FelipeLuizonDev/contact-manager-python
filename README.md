# Contact Manager

A simple command-line contact manager developed in **Python** as part of the Rocketseat Python learning journey.

## Features

* Add new contacts
* List all contacts
* Edit existing contacts
* Mark or unmark contacts as favorites
* List favorite contacts
* Delete contacts

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/FelipeLuizonDev/contact-manager-python.git
```

2. Navigate to the project folder:

```bash
cd contact-manager-python
```

3. Run the application:

```bash
python main.py
```
